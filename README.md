# push-baidu-link
百度链接推送
这是一款WordPress的插件，可以主动向百度推动新提交文章的链接。

因为暂时还没有提交到WordPress上，所以使用方法如下
将插件下载解压，放到你WordPress目录下的wp-content/plugins中，然后进入WordPress的插件中心进行安装。

错误修正：

1、修正部分链接提交错误问题。

2、修正了部分链接重复提交问题。

3、修正了输入框太短问题。

4、其它

新增功能：

1、在写文章时，可以选择是否推送本篇文章。
